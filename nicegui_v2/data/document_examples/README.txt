CARPETA DE EJEMPLOS DOCUMENTALES

Esta carpeta queda preparada para vincular ejemplos reales a cada requisito
de la biblioteca documental del sistema de gestion.

Convencion actual de nombres:

    <norma>__<requisito>.pdf

Ejemplos:

    iso_9001__4_4.pdf
    iso_14001__6_1_2.pdf
    iso_45001__8_2.pdf
    iatf_16949__8_3_5_3.pdf

Notas:
- La norma usa minusculas y espacios convertidos a "_".
- El requisito reemplaza "." por "_".
- La interfaz ya muestra la clave esperada en cada requisito.
- Cuando exista un archivo con ese nombre, luego se podra abrir directamente
  desde el boton "Ver ejemplo".
