# IDEAS Consulting V2

Versión paralela en `NiceGUI`, separada de la app actual en Streamlit.

## Ejecutar

Desde la raíz del proyecto:

```powershell
python nicegui_v2\app.py
```

La app levanta por defecto en:

```text
http://127.0.0.1:8502
```

## Qué incluye

- Inicio institucional
- Alta de empresas
- Nuevo diagnóstico
- Resultados ejecutivos
- Historial con acciones rápidas

## Qué reutiliza

- `ideas.db`
- `Data/diagnostico.xlsx`
- `logo.png`
- `ideas_home_banner.png`

## Nota

Esta V2 corre en carpeta separada y no modifica la app Streamlit existente.
